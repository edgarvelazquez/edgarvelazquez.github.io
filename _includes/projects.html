2-3 Tree

  • Java

Designed Generic 2-3 Tree (B-Tree of Factor of 3) as a viable replacement for Java’s AVL Tree, Supports insertion, searching, clearing, returning an iterator for in-order traversal of the tree and checking the size of the tree. A 2-3 Tree is better than Binary Tree in cases where data inserted in Binary tree may cause it to form a vine like structure. A 2-3 Tree is balanced, meaning all its leaves will be at the same level.
